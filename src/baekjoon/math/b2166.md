백준 2166번 **다각형의 면적** 문제는 2차원 좌표 평면 위에 주어진 다각형의 넓이를 구하는 아주 고전적인 기하학 문제입니다.

다각형이 볼록하든 오목하든 상관없이, 좌표가 순서대로 주어졌을 때 가장 쉽고 완벽하게 면적을 구할 수 있는 마법 같은 공식이 있습니다. 바로 **'신발끈 공식(Shoelace Formula)'**입니다!

### 💡 핵심 아이디어: 신발끈 공식

이름이 왜 신발끈 공식이냐면, 좌표들을 위아래로 나열해 놓고 대각선으로 곱하는 모양이 마치 신발끈을 묶는 모양과 같아서 붙여진 이름입니다.

1. $N$개의 점 좌표 $(x_1, y_1), (x_2, y_2), \dots, (x_N, y_N)$을 순서대로 나열합니다.
2. **마지막에 첫 번째 점 $(x_1, y_1)$을 한 번 더 적어줍니다.** (다각형을 닫아야 하니까요!)
3. **오른쪽 아래 대각선($\searrow$)** 방향으로 곱한 값들을 모두 더합니다. (`sum1`)
4. **왼쪽 아래 대각선($\swarrow$)** 방향으로 곱한 값들을 모두 더합니다. (`sum2`)
5. 두 합의 차이를 구하고 절댓값을 씌운 뒤, **2로 나누어 줍니다.**

$$\text{Area} = \frac{1}{2} | (x_1y_2 + x_2y_3 + \dots + x_Ny_1) - (y_1x_2 + y_2x_3 + \dots + y_Nx_1) |$$

### 1. Java 풀이

좌표의 최댓값이 100,000이므로, 두 좌표를 곱하면 $10^{10}$이 되어 **`int` 범위를 가볍게 초과(오버플로우)합니다.** 따라서 반드시 **`long` 타입**을 사용해야 합니다!

```java
import java.io.*;
import java.util.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        int N = Integer.parseInt(br.readLine());

        long[] x = new long[N + 1];
        long[] y = new long[N + 1];

        for (int i = 0; i < N; i++) {
            StringTokenizer st = new StringTokenizer(br.readLine());
            x[i] = Long.parseLong(st.nextToken());
            y[i] = Long.parseLong(st.nextToken());
        }

        // 마지막에 첫 번째 좌표를 한 번 더 넣어줍니다. (신발끈 공식의 핵심)
        x[N] = x[0];
        y[N] = y[0];

        long sum1 = 0;
        long sum2 = 0;

        // 대각선으로 곱해서 더하기
        for (int i = 0; i < N; i++) {
            sum1 += x[i] * y[i + 1]; // ↘ 방향 곱셈
            sum2 += y[i] * x[i + 1]; // ↙ 방향 곱셈
        }

        // 두 합의 차이의 절댓값을 구하고 2로 나눔 (소수점 첫째 자리까지 출력)
        double area = Math.abs(sum1 - sum2) / 2.0;
        
        // 소수점 첫째 자리까지 강제 출력 (예: 100.0)
        System.out.printf("%.1f\n", area);
    }
}

```

### 2. Python 풀이

파이썬은 정수 오버플로우 걱정이 없어서 아주 편하게 코드를 짤 수 있습니다.

```python
import sys
input = sys.stdin.readline

def solve():
    N = int(input())
    
    x = []
    y = []
    for _ in range(N):
        a, b = map(int, input().split())
        x.append(a)
        y.append(b)
        
    # 다각형을 닫기 위해 첫 번째 점을 마지막에 추가
    x.append(x[0])
    y.append(y[0])
    
    sum1 = 0
    sum2 = 0
    
    # 신발끈 공식 적용
    for i in range(N):
        sum1 += x[i] * y[i+1]
        sum2 += y[i] * x[i+1]
        
    area = abs(sum1 - sum2) / 2
    
    # 소수점 첫째 자리까지 출력
    print(f"{area:.1f}")

if __name__ == "__main__":
    solve()

```

### 🚨 반드시 체크해야 할 실수 포인트

1. **자료형 오버플로우 (Java):** 아까 강조했듯, 좌표가 최대 10만이라 $x_i \times y_{i+1}$은 최대 100억입니다. 자바의 `int`는 약 21억까지만 담을 수 있으므로 무조건 **`long`** 배열로 받아야 합니다.
2. **소수점 출력 포맷:** 면적이 정수로 딱 떨어져서 `100`이 나오더라도, 문제 조건에 따라 `100.0`으로 출력해야 합니다. Java의 `System.out.printf("%.1f\n", area)`나 Python의 `f"{area:.1f}"`를 사용하면 아주 깔끔하게 해결됩니다.
3. **마지막 점과 첫 점의 연결:** `N-1`번째 점과 `0`번째 점(처음 점)도 잊지 말고 꼭 교차 곱셈을 해주어야 합니다. 이를 편하게 하기 위해 배열 맨 끝에 0번째 좌표를 한 번 더 복사해 두는 테크닉이 아주 유용합니다.

신발끈 공식을 처음 보셨다면 대각선 곱셈 과정이 조금 신기하실 텐데, 혹시 수식이나 코드가 맞물려 돌아가는 과정 중에 더 설명이 필요한 부분이 있으신가요?
