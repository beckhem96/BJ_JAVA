이전에 질문하신 2636번 문제와 방금 질문하신 **2638번 치즈** 문제는 형제 문제입니다!
2636번은 공기와 **한 면**만 닿아도 녹지만, 2638번은 공기와 **두 면 이상** 닿아야 녹는다는 조건이 추가된 한 단계 업그레이드된 문제입니다.

이 문제들의 가장 큰 난관은 **"치즈 내부의 빈 공간(구멍)과 외부 공기를 어떻게 구분할 것인가?"**입니다.

### 💡 핵심 아이디어: "외부 공기" 기준의 BFS

치즈 입장에서 밖을 보려고 하면 구멍인지 밖인지 알기 어렵습니다. 발상을 전환해서 **"외부 공기 입장에서 치즈를 향해 BFS를 돌려야"** 합니다.

1. **가장자리는 항상 공기:** 문제 조건에서 모서리는 무조건 치즈가 없다고 했습니다. 따라서 `(0, 0)`은 무조건 외부 공기입니다.
2. **외부 공기 BFS:** `(0, 0)`에서 시작해서 치즈가 없는 빈 공간(`0`)만 골라서 탐색(BFS)을 냅니다. 이 탐색이 닿는 모든 `0`은 확실한 '외부 공기'가 됩니다. 치즈 내부의 구멍은 외부와 막혀 있으므로 이 BFS가 닿지 못합니다!
3. **접촉 횟수 세기:** 공기가 퍼져나가다가 치즈(`1`)를 만나면, 해당 치즈의 **접촉 횟수**를 1 증가시킵니다.
4. **치즈 녹이기:** BFS 탐색이 끝난 후, 전체 맵을 확인하여 접촉 횟수가 **2 이상**인 치즈들을 `0`(빈 공간)으로 바꿉니다.

### 1. Java 풀이

접촉 횟수를 기록할 `contact` 배열을 매 시간마다 초기화하여 사용하는 것이 핵심입니다.

```java
import java.io.*;
import java.util.*;

public class Main {
    static int N, M;
    static int[][] map;
    static int[] dr = {-1, 1, 0, 0};
    static int[] dc = {0, 0, -1, 1};
    static int cheeseCount = 0; // 남은 치즈 개수

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st = new StringTokenizer(br.readLine());

        N = Integer.parseInt(st.nextToken());
        M = Integer.parseInt(st.nextToken());
        map = new int[N][M];

        for (int i = 0; i < N; i++) {
            st = new StringTokenizer(br.readLine());
            for (int j = 0; j < M; j++) {
                map[i][j] = Integer.parseInt(st.nextToken());
                if (map[i][j] == 1) cheeseCount++;
            }
        }

        int time = 0;

        // 치즈가 다 녹을 때까지 반복
        while (cheeseCount > 0) {
            bfs(); // 외부 공기 탐색 및 치즈 접촉 카운트
            time++;
        }

        System.out.println(time);
    }

    static void bfs() {
        Queue<int[]> q = new LinkedList<>();
        boolean[][] visited = new boolean[N][M];
        int[][] contact = new int[N][M]; // 외부 공기 접촉 횟수

        // (0, 0)은 항상 외부 공기
        q.offer(new int[]{0, 0});
        visited[0][0] = true;

        while (!q.isEmpty()) {
            int[] cur = q.poll();
            int r = cur[0];
            int c = cur[1];

            for (int i = 0; i < 4; i++) {
                int nr = r + dr[i];
                int nc = c + dc[i];

                if (nr >= 0 && nr < N && nc >= 0 && nc < M) {
                    // 빈 공간(외부 공기)이면 큐에 넣고 계속 탐색
                    if (map[nr][nc] == 0 && !visited[nr][nc]) {
                        visited[nr][nc] = true;
                        q.offer(new int[]{nr, nc});
                    } 
                    // 치즈를 만나면 접촉 횟수 증가 (큐에 넣지는 않음!)
                    else if (map[nr][nc] == 1) {
                        contact[nr][nc]++;
                    }
                }
            }
        }

        // 탐색이 끝난 후, 2면 이상 접촉한 치즈를 녹임
        for (int i = 0; i < N; i++) {
            for (int j = 0; j < M; j++) {
                if (contact[i][j] >= 2) {
                    map[i][j] = 0; // 치즈가 녹아서 빈 공간이 됨
                    cheeseCount--; // 남은 치즈 개수 감소
                }
            }
        }
    }
}

```

### 2. Python 풀이

파이썬도 동일한 로직을 적용합니다.

```python
import sys
from collections import deque

input = sys.stdin.readline

def solve():
    N, M = map(int, input().split())
    board = []
    cheese_count = 0
    
    for i in range(N):
        row = list(map(int, input().split()))
        board.append(row)
        cheese_count += sum(row)
        
    dr = [-1, 1, 0, 0]
    dc = [0, 0, -1, 1]
    time = 0
    
    # 외부 공기 확인 및 치즈 녹이기 함수
    def bfs():
        nonlocal cheese_count
        q = deque([(0, 0)])
        visited = [[False] * M for _ in range(N)]
        contact = [[0] * M for _ in range(N)]
        
        visited[0][0] = True
        
        while q:
            r, c = q.popleft()
            
            for i in range(4):
                nr = r + dr[i]
                nc = c + dc[i]
                
                if 0 <= nr < N and 0 <= nc < M:
                    # 빈 공간(외부 공기) 연쇄 탐색
                    if board[nr][nc] == 0 and not visited[nr][nc]:
                        visited[nr][nc] = True
                        q.append((nr, nc))
                    # 치즈를 만나면 접촉 카운트 증가
                    elif board[nr][nc] == 1:
                        contact[nr][nc] += 1
                        
        # 2번 이상 닿은 치즈 녹이기
        for i in range(N):
            for j in range(M):
                if contact[i][j] >= 2:
                    board[i][j] = 0
                    cheese_count -= 1

    # 모든 치즈가 녹을 때까지 반복
    while cheese_count > 0:
        bfs()
        time += 1
        
    print(time)

if __name__ == "__main__":
    solve()

```

### 🔍 풀이 포인트

1. **치즈를 만나면 큐에 넣지 않는다:** BFS는 오직 '외부 공기(`0`)'만을 타고 번져야 합니다. 치즈(`1`)를 만나면 접촉 횟수(`contact`)만 올려주고 탐색은 거기서 멈춰야 외부 공기가 치즈 내부 구멍으로 침투하는 것을 막을 수 있습니다.
2. **동시 다발적 녹음:** 치즈는 한 시간에 동시에 녹아야 합니다. 따라서 공기를 닿자마자 바로 `0`으로 바꿔버리면, 아직 이번 시간에 계산되지 않은 다른 치즈가 새롭게 구멍 뚫린 공기의 영향을 받게 됩니다. 반드시 **카운트를 다 세어둔 뒤 마지막에 일괄적으로 지워야 합니다.**
