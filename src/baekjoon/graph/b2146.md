백준 2146번 **다리 만들기** 문제는 BFS(너비 우선 탐색)를 두 번의 단계로 나누어 완벽하게 조합해야 풀 수 있는 아주 매력적인 문제입니다.

이 문제의 핵심은 **"어떤 섬에서 출발해서, 다른 어떤 섬에 도착했는지"**를 컴퓨터가 알아먹을 수 있게 구분해 주는 것입니다.

### 💡 핵심 아이디어: 2단계 BFS 전략

모든 육지가 똑같은 `1`로 되어 있으면 바다를 건너다 다른 육지를 만났을 때, "이게 원래 내 섬인지, 아니면 도착해야 할 다른 섬인지" 구분할 수 없습니다. 따라서 다음 2단계를 거쳐야 합니다.

1. **1단계: 섬에 이름(번호) 붙이기 (Grouping)**
* 지도를 쭉 훑으면서 육지(`1`)를 발견하면 BFS를 돌려 그 섬 전체를 고유한 번호(2, 3, 4...)로 칠해버립니다.
* 이렇게 하면 '2번 섬', '3번 섬'처럼 서로를 완벽하게 구별할 수 있습니다.


2. **2단계: 최단 거리 다리 놓기 (Shortest Bridge)**
* 각 섬(예: 2번 섬)에서 출발하는 BFS를 돌립니다.
* 2번 섬의 모든 땅을 큐(Queue)에 한 번에 다 넣고 바다(`0`)를 향해 한 칸씩 다리를 놓으며 전진합니다.
* 가다가 **나와 번호가 다른 땅(육지)을 만나는 순간!** 그 거리가 바로 다른 섬과 연결되는 최단 다리 길이가 됩니다.

### 1. Java 풀이

```java
import java.io.*;
import java.util.*;

public class Main {
    static int N;
    static int[][] map;
    static boolean[][] visited;
    static int[] dr = {-1, 1, 0, 0};
    static int[] dc = {0, 0, -1, 1};

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        N = Integer.parseInt(br.readLine());
        map = new int[N][N];
        visited = new boolean[N][N];

        for (int i = 0; i < N; i++) {
            StringTokenizer st = new StringTokenizer(br.readLine());
            for (int j = 0; j < N; j++) {
                map[i][j] = Integer.parseInt(st.nextToken());
            }
        }

        // 1단계: 섬 번호 붙이기 (2부터 시작)
        int islandId = 2;
        for (int i = 0; i < N; i++) {
            for (int j = 0; j < N; j++) {
                if (map[i][j] == 1 && !visited[i][j]) {
                    markIsland(i, j, islandId);
                    islandId++;
                }
            }
        }

        // 2단계: 각 섬에서 다른 섬으로 가는 최단 거리 찾기
        int minBridge = Integer.MAX_VALUE;
        for (int i = 2; i < islandId; i++) {
            minBridge = Math.min(minBridge, buildBridge(i));
        }

        System.out.println(minBridge);
    }

    // 1단계: BFS로 연결된 육지를 같은 번호로 마킹
    static void markIsland(int r, int c, int id) {
        Queue<int[]> q = new LinkedList<>();
        q.offer(new int[]{r, c});
        visited[r][c] = true;
        map[r][c] = id;

        while (!q.isEmpty()) {
            int[] cur = q.poll();
            for (int i = 0; i < 4; i++) {
                int nr = cur[0] + dr[i];
                int nc = cur[1] + dc[i];

                if (nr >= 0 && nr < N && nc >= 0 && nc < N) {
                    if (map[nr][nc] == 1 && !visited[nr][nc]) {
                        visited[nr][nc] = true;
                        map[nr][nc] = id;
                        q.offer(new int[]{nr, nc});
                    }
                }
            }
        }
    }

    // 2단계: 특정 섬(id)에서 바다를 건너 다른 섬을 찾는 BFS
    static int buildBridge(int id) {
        Queue<int[]> q = new LinkedList<>();
        int[][] dist = new int[N][N];

        // 거리 배열을 -1로 초기화하고, 현재 탐색할 섬의 땅을 큐에 모두 넣음
        for (int i = 0; i < N; i++) {
            Arrays.fill(dist[i], -1);
            for (int j = 0; j < N; j++) {
                if (map[i][j] == id) {
                    q.offer(new int[]{i, j});
                    dist[i][j] = 0; // 시작점(육지)은 다리 길이 0
                }
            }
        }

        while (!q.isEmpty()) {
            int[] cur = q.poll();
            int r = cur[0];
            int c = cur[1];

            for (int i = 0; i < 4; i++) {
                int nr = r + dr[i];
                int nc = c + dc[i];

                if (nr >= 0 && nr < N && nc >= 0 && nc < N) {
                    // 1. 바다(0)를 만나면 다리를 놓으며 전진
                    if (map[nr][nc] == 0 && dist[nr][nc] == -1) {
                        dist[nr][nc] = dist[r][c] + 1;
                        q.offer(new int[]{nr, nc});
                    }
                    // 2. 다른 섬을 만났다면! (바다도 아니고, 내 섬도 아님)
                    else if (map[nr][nc] > 0 && map[nr][nc] != id) {
                        return dist[r][c]; // 현재까지 놓은 다리 길이를 즉시 리턴
                    }
                }
            }
        }
        return Integer.MAX_VALUE;
    }
}

```

### 2. Python 풀이

```python
import sys
from collections import deque

input = sys.stdin.readline

def solve():
    N = int(input())
    board = [list(map(int, input().split())) for _ in range(N)]
    visited = [[False] * N for _ in range(N)]
    
    dr = [-1, 1, 0, 0]
    dc = [0, 0, -1, 1]
    
    # 1단계: 섬 번호 마킹하기
    def mark_island(r, c, island_id):
        q = deque([(r, c)])
        visited[r][c] = True
        board[r][c] = island_id
        
        while q:
            curr_r, curr_c = q.popleft()
            
            for i in range(4):
                nr = curr_r + dr[i]
                nc = curr_c + dc[i]
                
                if 0 <= nr < N and 0 <= nc < N:
                    if board[nr][nc] == 1 and not visited[nr][nc]:
                        visited[nr][nc] = True
                        board[nr][nc] = island_id
                        q.append((nr, nc))

    # 섬 번호는 2부터 시작 (기존의 1, 0과 겹치지 않게)
    island_id = 2
    for i in range(N):
        for j in range(N):
            if board[i][j] == 1 and not visited[i][j]:
                mark_island(i, j, island_id)
                island_id += 1

    # 2단계: 각 섬에서 다른 섬으로 가는 최단 거리 구하기
    def build_bridge(target_id):
        q = deque()
        dist = [[-1] * N for _ in range(N)]
        
        # 현재 섬(target_id)의 모든 좌표를 큐에 넣음
        for i in range(N):
            for j in range(N):
                if board[i][j] == target_id:
                    q.append((i, j))
                    dist[i][j] = 0
                    
        while q:
            r, c = q.popleft()
            
            for i in range(4):
                nr = r + dr[i]
                nc = c + dc[i]
                
                if 0 <= nr < N and 0 <= nc < N:
                    # 바다를 만나면 거리 증가 후 큐에 추가
                    if board[nr][nc] == 0 and dist[nr][nc] == -1:
                        dist[nr][nc] = dist[r][c] + 1
                        q.append((nr, nc))
                    # 다른 섬을 만났다면 최단 거리 반환
                    elif board[nr][nc] > 0 and board[nr][nc] != target_id:
                        return dist[r][c]
                        
        return float('inf')

    # 모든 섬에 대해 최단 다리 길이 계산 후 최솟값 찾기
    min_ans = float('inf')
    for i in range(2, island_id):
        min_ans = min(min_ans, build_bridge(i))
        
    print(min_ans)

if __name__ == "__main__":
    solve()

```

### 🔍 반드시 체크해야 할 실수 포인트

1. **섬 번호를 2부터 시작하는 이유:** * 입력에서 이미 육지를 1, 바다를 0으로 주고 있습니다. 섬 번호를 1부터 매기면, 아직 탐색하지 않은 육지(1)와 이미 번호가 매겨진 1번 섬(1)이 구별되지 않아 무한 루프나 탐색 오류에 빠집니다.
2. **`dist` 배열의 초기화 (`-1`):**
* 거리를 저장하는 배열은 다리가 없는 상태를 0으로 쓰면 안 됩니다. 출발점인 육지 자체가 0초(다리 길이 0)이기 때문입니다. 따라서 아직 방문하지 않은 바다는 반드시 -1로 초기화해야 정상적으로 길이를 더해 나갈 수 있습니다.

### 1. `visited` 배열의 역할을 `dist` 배열이 대신하고 있습니다.

따로 `boolean[][] visited`를 만들지 않았을 뿐, `dist`(거리) 배열이 그 역할을 완벽하게 수행하고 있습니다.

코드의 이 부분을 다시 볼까요?

```java
// 1. 초기화: 모든 바다의 거리를 -1(미방문)로 세팅
for (int i = 0; i < N; i++) {
    Arrays.fill(dist[i], -1); 
    // ... (생략) ...
}

// 2. 큐에 넣을 때의 조건문
if (map[nr][nc] == 0 && dist[nr][nc] == -1) { 
    dist[nr][nc] = dist[r][c] + 1; // -1에서 1 이상의 숫자로 바뀜 (방문 처리 됨!)
    q.offer(new int[]{nr, nc});
}

```

* **`dist[nr][nc] == -1`의 의미:** "여기는 아직 한 번도 도착한 적 없는 바다야!" 즉, **`!visited`와 완전히 똑같은 역할**을 합니다.
* 큐에 넣으면서 `dist` 값을 1, 2, 3...으로 바꿔주기 때문에, 다음에 다른 경로로 이 바다에 오더라도 `dist`가 더 이상 `-1`이 아니므로 `pass`하게 됩니다. (중복 방문 X)

### 2. 시작점을 큐에 몽땅 넣는 이유 (다중 시작점 BFS)

"섬의 모든 좌표를 다 큐에 넣으면 비효율적이지 않을까?"라는 생각이 드실 수 있습니다. 하지만 이는 **다중 시작점 BFS (Multi-source BFS)**라는 아주 강력하고 효율적인 테크닉입니다.

* **만약 섬의 테두리 1칸에서만 BFS를 시작한다면?**
  섬의 왼쪽 끝에서 출발한 BFS와 오른쪽 끝에서 출발한 BFS를 각각 따로 돌려서 최솟값을 비교해야 합니다. (엄청난 시간 낭비)
* **모든 좌표를 한 번에 다 넣고 시작하면?**
  큐에 들어있는 섬의 모든 칸들이 **"동시에 1칸씩 바다를 향해 팽창(전진)"**하게 됩니다. 아메바가 커지는 모습을 상상하시면 됩니다.
  이렇게 섬 전체가 동시에 1칸씩 살이 찌다가, **가장 먼저 다른 섬에 닿는 촉수(경로)가 무조건 전체 섬을 통틀어 최단 거리**가 됩니다!

### 💡 요약

1. `dist == -1` 조건이 이미 갔던 바다를 다시 가지 않게 완벽하게 막아줍니다.
2. 내 섬의 모든 땅을 큐에 몽땅 넣고 시작하는 것은 낭비가 아니라, **"섬 전체를 한 번에 1칸씩 팽창시켜 가장 빠른 지름길을 한 방에 찾기 위한"** 최고의 효율적인 방법입니다.
