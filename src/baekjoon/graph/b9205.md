백준 9205번 **맥주 마시면서 걸어가기** 문제는 재미있는 스토리 속에 **그래프 탐색(BFS 또는 DFS)**의 원리가 숨어있는 전형적인 도달 가능성(Reachability) 확인 문제입니다.

가장 중요한 포인트는 **"맥주 상자의 룰을 그래프의 간선(Edge) 조건으로 변환하는 것"**입니다.

### 💡 핵심 아이디어: "1000미터 점프"

1. **이동 가능 거리의 한계:**
* 맥주는 최대 20병까지 들 수 있고, 50미터에 한 병씩 마셔야 합니다.
* 즉, 한 번에 이동할 수 있는 최대 거리는 $20 \times 50 = 1000$미터입니다.
* 편의점에 도착하면 맥주를 다시 20병으로 꽉 채울 수 있으므로, **"현재 위치에서 다음 위치(편의점 또는 페스티벌)까지의 거리가 1000미터 이하인가?"**가 핵심 조건이 됩니다.


2. **맨해튼 거리 (Manhattan Distance):**
* 문제에서 거리는 `x 좌표의 차이 + y 좌표의 차이`로 계산하라고 명시되어 있습니다.
* 거리 공식: $|x_1 - x_2| + |y_1 - y_2|$


3. **BFS 탐색 로직:**
* 집(출발점)을 큐에 넣고 시작합니다.
* 현재 위치에서 **페스티벌(도착점)까지 1000미터 이내라면?** $\rightarrow$ 바로 "happy" 출력 후 종료!
* 그렇지 않다면, **아직 방문하지 않은 편의점들 중 1000미터 이내에 있는 곳**을 모두 큐에 넣고 방문 처리합니다.
* 큐가 다 빌 때까지 페스티벌에 도착하지 못했다면 $\rightarrow$ "sad" 출력.

### 1. Java 풀이

좌표를 저장하기 위해 간단한 `Point` 클래스(또는 int 배열)를 활용하여 BFS를 구현합니다. 여러 개의 테스트 케이스가 주어지므로 변수 초기화에 주의해야 합니다.

```java
import java.io.*;
import java.util.*;

public class Main {
    static class Point {
        int x, y;
        public Point(int x, int y) {
            this.x = x;
            this.y = y;
        }
    }

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringBuilder sb = new StringBuilder();
        
        int t = Integer.parseInt(br.readLine()); // 테스트 케이스 개수
        
        while (t-- > 0) {
            int n = Integer.parseInt(br.readLine()); // 편의점 개수
            
            // 집 좌표
            StringTokenizer st = new StringTokenizer(br.readLine());
            Point house = new Point(Integer.parseInt(st.nextToken()), Integer.parseInt(st.nextToken()));
            
            // 편의점 좌표들
            Point[] stores = new Point[n];
            for (int i = 0; i < n; i++) {
                st = new StringTokenizer(br.readLine());
                stores[i] = new Point(Integer.parseInt(st.nextToken()), Integer.parseInt(st.nextToken()));
            }
            
            // 페스티벌 좌표
            st = new StringTokenizer(br.readLine());
            Point festival = new Point(Integer.parseInt(st.nextToken()), Integer.parseInt(st.nextToken()));
            
            // BFS 탐색
            if (bfs(house, stores, festival, n)) {
                sb.append("happy\n");
            } else {
                sb.append("sad\n");
            }
        }
        System.out.print(sb);
    }

    static boolean bfs(Point house, Point[] stores, Point festival, int n) {
        Queue<Point> q = new LinkedList<>();
        boolean[] visited = new boolean[n]; // 편의점 방문 여부
        
        q.offer(house);
        
        while (!q.isEmpty()) {
            Point cur = q.poll();
            
            // 1. 현재 위치에서 페스티벌까지 한 번에 갈 수 있는지 확인
            if (Math.abs(cur.x - festival.x) + Math.abs(cur.y - festival.y) <= 1000) {
                return true;
            }
            
            // 2. 갈 수 없다면, 갈 수 있는(1000m 이내) 미방문 편의점을 찾음
            for (int i = 0; i < n; i++) {
                if (!visited[i]) {
                    int dist = Math.abs(cur.x - stores[i].x) + Math.abs(cur.y - stores[i].y);
                    if (dist <= 1000) {
                        visited[i] = true;
                        q.offer(stores[i]);
                    }
                }
            }
        }
        
        return false; // 큐가 빌 때까지 페스티벌에 도달하지 못함
    }
}

```

### 2. Python 풀이

파이썬은 튜플이나 리스트를 사용하여 좌표를 다루면 코드가 매우 간결해집니다.

```python
import sys
from collections import deque

input = sys.stdin.readline

def solve():
    t = int(input())
    
    for _ in range(t):
        n = int(input()) # 편의점 개수
        
        hx, hy = map(int, input().split()) # 집 좌표
        stores = [tuple(map(int, input().split())) for _ in range(n)] # 편의점 좌표
        fx, fy = map(int, input().split()) # 페스티벌 좌표
        
        visited = [False] * n
        q = deque([(hx, hy)])
        is_happy = False
        
        while q:
            cx, cy = q.popleft()
            
            # 1. 페스티벌까지 바로 갈 수 있다면 종료
            if abs(cx - fx) + abs(cy - fy) <= 1000:
                is_happy = True
                break
                
            # 2. 1000m 이내에 있는 방문하지 않은 편의점을 큐에 추가
            for i in range(n):
                if not visited[i]:
                    nx, ny = stores[i]
                    if abs(cx - nx) + abs(cy - ny) <= 1000:
                        visited[i] = True
                        q.append((nx, ny))
                        
        if is_happy:
            print("happy")
        else:
            print("sad")

if __name__ == "__main__":
    solve()

```

### 🔍 반드시 체크해야 할 주의 포인트

1. **편의점 방문 처리 (`visited` 배열):**
* 만약 방문 처리를 하지 않으면, A 편의점과 B 편의점 사이의 거리가 1000미터 이내일 때 두 편의점을 무한히 왔다 갔다 하는 **무한 루프**에 빠지게 됩니다. (맥주는 계속 리필되니까요!)
* 따라서 한 번 거쳐 간 편의점은 다시 갈 필요가 없으므로 확실하게 `visited` 처리를 해줘야 합니다.


2. **도착지(페스티벌)도 매번 거리를 재야 합니다:**
* 집이나 특정 편의점에서 편의점만 바라보는 것이 아니라, **현재 내 위치에서 페스티벌로 곧바로 쏠 수 있는지**(`거리 <= 1000`)를 큐에서 꺼낼 때마다 제일 먼저 확인하는 것이 핵심입니다.



이전 문제들에서 2차원 배열 격자 칸을 한 칸씩 이동하는 BFS를 주로 보셨다면, 이 문제는 **"특정 좌표 간의 텔레포트(점프)"**를 간선으로 삼는 BFS입니다.
